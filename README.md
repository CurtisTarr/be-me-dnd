# be-me-dnd
Quick DnD 5E random character generator

## Requirements
- Python 3

## Running
- Clone repo
- Open terminal
- Navigate to the repo
- Run `python3 be_me_dnd.py`

## Example output
```
> Be me Elmo Slomkowski
> Half-Orc Paladin
> {'Strength': 11, 'Dexterity': 11, 'Constitution': 4, 'Intelligence': 9, 'Wisdom': 7, 'Charisma': 13}
```
