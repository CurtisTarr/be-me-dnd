# be-me-dnd
Quick DnD 5E random character generator

## Requirements
- Python 3

## Running
1. Clone repo
2. Open terminal
3. Navigate to the repo
4. Run `python3 be_me_dnd.py`

## Example output
```
> Be me Sammie Freeburger
> Halfling Paladin
> Strength: 4
> Dexterity: 6
> Constitution: 5
> Intelligence: 9
> Wisdom: 13
> Charisma: 5
```
